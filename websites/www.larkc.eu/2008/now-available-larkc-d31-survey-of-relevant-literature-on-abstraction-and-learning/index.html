NOW AVAILABLE: LarKC D3.1 Survey of relevant literature on abstraction and learning

Posted October 3rd, 2008

Download of LarKC D3.1 Survey of relevant literature on abstraction and learning

The first part of this report explores the challenges for machine learning on the Semantic Web, and the opportunities generated by these challenges.

The second part of the report looks at opportunities for stream-reasoning in Semantic Web use-cases. It proposes a continuous query language for the Semantic Web (named C-SPARQL). Such continuous query language extends SPARQL with data streams, aggregation operators, and a continuous semantics.

FOCUS GROUP:
First part:

  • Researchers in Machine Learning with an interest in Semantic
    Web opportunities.

Second part:

  • Researchers in (streaming) Database-technology with an interest in
    Semantic Web opportunities,
  • developers of RDF technology, with an interest to dealing with
    streaming data
