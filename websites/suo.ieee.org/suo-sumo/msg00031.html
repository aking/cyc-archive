Allan,
   We should clarify that the "old GNU/Teknowledge" license did not exist
on prior versions.  I agree that the GNU license was not appropriate.  The
versions that I released when I was running the project did not have the
"old GNU/Teknowledge" license, nor the new IEEE one.  My understanding was
that copyright belonged to IEEE and no further license was necessary.  All
versions that were submitted for a vote by the SUO list did not have a
license embedded.

Adam
