Changes from 2001-12-05 to 2002-01-02

WHAT HAS NOT CHANGED:

  • The Introduction.htm

WHAT HAS BEEN MODIFIED:

  • The IFF Basic KIF Ontology has been slightly modifed. This is the interface ontology for the KIF language containing generic concepts. It has only one namespace with recommended prefix 'KIF'. The only changes to the IFF Basic Ontology is in Tables 1, 2 and 4 (mainly 2).

  • The IFF Core Ontology has been slighlty modified. The only changes to the Core Ontology is the addition of the order namespaces ('ORD', 'ORD.FTN', 'ORD.ADJ', 'ORD.REL', 'ORD.CLO') and graph namespaces ('GPH' and 'GPH.MOR'). Of course, these add a lot to the size.

  • The IFF Classification Ontology has been updated to accord with the Basic KIF Ontology. It has also lost its order namespace to the Core Ontology.

  • The IFF Category Theory Ontology has been updated to accord with the Basic KIF Ontology. It has also lost its graph namespaces to the Core Ontology.

WHAT IS NEW:

  • No new documents at this time
