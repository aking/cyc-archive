The IFF Generic Component

  • IFF-INS

The generic component of the applied aspect of the natural part of the IFF represents generic applications. Currently, there is only one meta-ontology in the generic component — the generic and parametric IFF Institution Theory (meta) Ontology (IFF-INS). Generic means that the terminology and axiomatization for any two metalevels is identical. Parametric means that the metalevel index is a parameter. Hence, only one copy of the generic IFF-INS meta-ontology with a level parameter is needed for all finite levels.
