Changes to IFF from previous version to current version (October 4, 2001)

    

  •  What was the introduction section in the starter document 
     is now a separate Introduction file.

    

  •  What was the "Large Aspect" in the starter document
     is now _mostly_ in the IFF Core (sub)Ontology file.
     The exception is that the "Large Orders" subsection of the starter document
     is now in the IFF Classification (sub)Ontology file.

    

  •  What was the Category Theory Ontology section in the starter document
     is now _mostly_ in the IFF Category Theory (sub)Ontology file.
     The exceptions are that the the "Namespace of Large Classifications"
     and the "Namespace of Large Concept Lattices" are now in the
     IFF Classification (sub)Ontology file.

    

  •  The "Large Aspect" has thus expanded and now includes 
     (1) the IFF Core (sub)Ontology;
     (2) the IFF Category Theory (sub)Ontology; and
     (3) the IFF Classification (sub)Ontology.

    

  •  What was the model theory ontology section in the starter document
     is now not covered by an updated document.
     We expect this document to be posted some time in the coming months.
