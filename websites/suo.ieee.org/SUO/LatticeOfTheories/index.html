Lattice of Theories


Search IEEE

This work was approved by resolution 3 of the SUO WG.

The components of this contribution are:

  1. Develop a Lattice of Theories based on (but not limited to):
    • IFF
    • SUMO
    • OpenCYC
    • SUO 4D Ontology
  2. Develop a Lattice of Theories as a set of modules where:
    • Each module consists of a theory together with documentation and other metadata.
    • The theory consists of axioms and definitions stated in a logic-based language, such as those in the Common Logic (CL) framework.
    • The Lattice of Theories is organized as a collection of modules, related in a generalization/specialization hierarchy.
  3. Develop a Register (a database) where the Lattice of Theories can be stored and queried. The Register provides facilities to store relations and axioms as well as concepts, and map different approaches to conceptualisation to each other.
  4. Develop a methodology for testing the consistency of the modules.

Work on some of these elements is underway.
