Bernard,
   This looks like an effort we should be connected with.  I already had 
references to all their sources on the SUO "Candidate Content" page.  Also, 
they've limited themselves to English definitions and binary relations - no 
axioms - which may make the work more "accessible" but not useful for 
automated reasoning.  Their main paper does say they intend to address move 
expressive representations though.  The goals of the project seem different 
from the SUO in that - from their main paper 
(http://www.sirma.bg/ontotext/publications/TR1OntoMap.pdf ) - "OntoMap is 
trying to use the minimal useful set of primitives."  It would be good if 
we could induce one member from the OntoMap project to participate in the 
SUO.  I've added a link to OntoMap from http://suo.ieee.org/links.html

Adam
