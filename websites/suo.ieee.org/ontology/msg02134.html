We continue with the beating of this still kicking horse that is
known as the propositional equation "(p (q))(p (r)) = (p (q r))".

I take up the "third way" that I mentioned for tasting such equations,
but I believe that you will be relieved to know that it is literally
a third way only at the very outset, almost immediately splitting up
according to whether one continues model-wise or proof-wise thereon.

To start at the starting blocks -- as I naturally like to do,
no matter where I fancy or foresee I might end up in the end --
let us begin with the observation that we are perfectly free
to express this equation, so far viewed as a metaproposition,
within the very object language that we have been expounding
all this time.  Indeed, it is one of the nicer properties of
any adequate propositional language in relation to the order
of abstract propositions that is the object of its demeanors
that "every proposition about the order is a propostion 'in'
the order".  Of course, this kind of auto-reflective closure
is achievable only because the order of propositions 'about'
the possible orderings of the object propositions themselves
is so overweeningly modest and so radically restrained about
its initial aims and its ultimate scope from the very outset,
being limited in the propositions that it hopes to entertain
about the order to which they belong to propositions such as
these:  "p equals q", or "p implies q", or their derivatives.
Moreover, it is one of the nicer features of this particular
syntax for a propositional calculus -- indeed, it was one of
the intentional features that was foremost on my mind at the
time of designing the "Cactus Language" that I use in RefLog --
that these sorts of equations and implications between pairs
of propositions are amenable to being so compactly expressed
and so fluently evaluated, by human, by machine, and at will.

Et so tuit!

Let us exchange for our propositional equation,
to wit, this one:  "(p (q))(p (r)) = (p (q r))",
the corresponding equational proposition, to wit,
this very one:  "(( (p (q))(p (r)) , (p (q r)) ))".
I'm sure that you would prefer to see it in pictures:

¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤
|
|               q o   o r   q o r
|                 |   |       |
|               p o   o p   p o
|                  \ /        |
|                   o---------o
|                    \       /
|                     \     /
|                      \   /
|                       \ /
|                        o
|                        |
|                        |
|                        |
|                        |
|                        @
|
| ((  (p (q))  (p (r))   ,   (p  (q r))  ))
|
|    [[p=>q] & [p=>r]]  <=>  [p=>[q&r]]
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤

I will leave you to contemplate that unturneddown glass,
while I go russell up some breakfast and maybe a mimosa!

Cheers, Hic Et Nunc,

Jon Awbrey
