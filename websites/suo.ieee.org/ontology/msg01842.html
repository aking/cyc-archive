At 25-03-01 09:33, Chris Menzel wrote:

I wrote:
> [Matthew wrote:]
> > If classes are non-extensional then I don't see what point there is in
> > having null.
>
> Nothing in the axioms entails that there is a null class.

This is false, of course.  According to the axioms, every class has a
complement, hence (complement top) is null. 

### No woman overboard, but note one can only prove this from the axioms if both top and null are known to be unique. There is as yet no axiom or meta axiom to this effect (nor is it provable of course).

Still, without
extensionality, if you want to have a name for a distinguished null class,
you need to introduce an axiom to that effect.
