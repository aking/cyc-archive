LOR.  Note 49

o~~~~~~~~~o~~~~~~~~~o~~~~~~~~~o~~~~~~~~~o~~~~~~~~~o

We come now to the very special cases of 2-adic relations
that are classified as functions.  It will serve a double
duty toward the work of exposition if we use the class of
functions as object examples to clarify the more abstruse
definitions in the RIG material.

To begin, let's recall the definition of a local flag:

L_x@j  =  {<x_1, ..., x_j, ..., x_k> in L : x_j = x}.

In the case of a 2-adic relation L c X_1 x X_2 = X x Y,
we can derive the benefits of a radical simplification
the definitions of the local flags.  Also in this case,
we tend to denote L_u@1 by "L_u@X" and L_v@2 by "L_v@Y".

Thus, for a 2-adic relation L c X x Y, we
have these definitions of the local flags:

L_u@X  =  {<x, y> in L : x = u}

       =  the set of all ordered pairs in L incident with u in X.

L_v@Y  =  {<x, y> in L : y = v}

       =  the set of all ordered pairs in L incident with v in Y.

A sufficient illustration is supplied by the earlier example E.

o   o   o   o   o   o   o   o   o   o   X

       \|/ \|  \   \   \|   |  \
o   o   o   o   o   o   o   o   o   o   Y
