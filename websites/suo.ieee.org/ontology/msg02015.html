Also, let me remind you of this
brand of distribution principle:

| An equation of the form "x(y) = (xy)" is NOT allowed, of course,
| but an equation of the following form is valid whenever k > 1.
|
|   x ( y1, ..., yk )  =  ( x y1, ..., x yk )
|
| To see this, if x is true, then it makes no difference on either
| side of the equation, but if x is false, then both sides are false.
| There can be only one, remember.

Now, the argument that I just gave for this form of distributive law
depended on the truth-functional interpretation of the lobe operator,
so this marks a patch that will need to be fixed up at a later point.

Anyway, these two facts in addition to the rest,
should be enough to see the proof of this truth:

---------------------------------------------------
 Case Analysis Principle (CAP)
 Case Analysis Theorem   (CAT)
---------------------------------------------------

|                                         o
|                                     x   |
|                              Q[o/x] o---o Q[|/x]
|         Q                            \ /
|         @              =              @
|
