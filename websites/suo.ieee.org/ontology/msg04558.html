Here are some things that come to mind as we begin to think about
the "ontological correlates" of objects, signs, and interpretants,
leading to the consideration of the usual sign relational domains
O, S, I as "ontological hierarchies" in their own rights.

Figure 1 presents a generic picture of this type of situation.

|               ^ ^ ^             :   v v v                 |
|                \|/           :     o  o  o  Instances     |

Figure 3 gives a picture of one way that an indexical sign functions,
denoting its objects by virtue of instances that it shares with them.
I have not convinced myself one way or the other whether this befits
indices in general, but it does account for how indexical signs work
in a particular type of dynamic or systems-theoretic setting.  Here,
the thing that we call an object corresonds to a trajectory through
a state space that is extended over a range of "instants" in time.
But the state of the total system has aspects or components that
project as the state of the object system O, the state of the
sign system S, and the state of the interpretent system I,
as in this setting systems have come to fill the roles of
the sign relational domains.  In this type of situation,
the usual description of the indexical sign as one that
bears an actual connection with its object amounts to
the statement that there is a common instance of the
total system state that projects at the same time to
the object state and to the sign state in question.
