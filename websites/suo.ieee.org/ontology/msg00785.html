Dear Mike,

If you read ISO18876-1 (www.iso18876.org) you will see that this is
recognised. Both sorts of model/ontology are supported with the necessary
mappings between them. 

So it isn't a matter of having to trade off, but of using different ones
appropriately. We find that the integrating one has the effect of defining
the more specfic ones through the mapping, which actually performs a
additional useful analysis function.
