I think we can all agree that we want to talk about the real world. 
That much is easy to agree on. However, I get the impression that 
some of the contributors to these discussions feel that this 
agreement somehow solves, or avoids, all the tricky problems. (After 
all, people get along in the real world talking to one another about 
it (in English, more or less), so why don't we all just kind of go on 
doing that for the SUO?) The trouble is that we have to decide HOW to 
describe this real world, and there is more than one way to do it. In 
fact there are probably hundreds of ways to do it.

John Sowa rightly wants us to avoid the tar-pit of endless 
metaphysical discussions about the nature of relationships, etc.; but 
that is only one of the places we might get stuck. We can avoid a lot 
of metaphysics, but we can't avoid the actual ontology. There is a 
lot more philosophical discussion about how to describe the world 
which we cannot avoid. Naive realism isnt going to help us here. It 
is going to take time to wade through some of this stuff.
