On Sun, 25 Mar 2001, Robert Meersman wrote:

> At 25-03-01 09:33, Chris Menzel wrote:
>
> >I wrote:
> > > [Matthew wrote:]
> > > > If classes are non-extensional then I don't see what point there is in
> > > > having null.
> > >
> > > Nothing in the axioms entails that there is a null class.
> >
> >This is false, of course.  According to the axioms, every class has a
> >complement, hence (complement top) is null.
>
> ### No woman overboard, but note one can only prove this from the
> axioms if both top and null are known to be unique. There is as yet no
> axiom or meta axiom to this effect (nor is it provable of course).

I think you are confusing the property *null* (i.e., the property of
having no instances) with the class *nil*, Robert.  The conclusion says
only that (complement top) has the property of being null, not that it is
identical with the class nil.  Since everything is an instance of top
(axiom), nothing is an instance of (complement top) (axiom for
complement), i.e., it is null (axiom for null).

-chris
