> >Right.  My initial position would be that the ontology should take 
> >an intentional view of terms, that disjointness should have a model 
> >force of "necessary" (Bill Andersen pointed this out to me in a 
> >private email) and therefore the axiom should be simple implication.
> 
> That decision would be a complete and total disaster. It would
> immediately make the structural ontology unusable, if taken seriously.
> It would be require that KIF be replaced by a modal language and
> render all the axioms first-order invalid. 

Not that I think a lot hangs on this, but I don't get it.  To change the
disjointness axiom so it uses only a conditional would simply allow
models where classes could have no instances in common but still not be
disjoint.  At which point, exactly, does it start raining fire and
brimstone?

> And moreover, it would be based on a stupid misunderstanding of the
> nature of modal truth - sorry, but I have tried to explain this
> politely to you guys on a number of occassions.

Sheesh, Pat, if that was polite, I'd hate to see you on a rampage!
