We have been looking at various ways of transforming
propositional expressions, both textual and visual,
while preserving certain aspects of their meaning,
to start with that vaguest of all possible words,
merely as a promissory note, to be cashed out in
a specified species of currency at a later date.

I cannot pretend acquaintance with or comprehension of all forms
of "intensions" (properties) that others might find of interest
in a given form of expression, nor can I speak for every form
of meaning that another might find of significance in a given
form of syntax.  The best that I can do is to try to specify
what my object is in using these propositional expressions,
and, on a related note, what aspects of their syntax I find
to be pertinent toward that object, and thus of concern to
preserve as I put these expressions through the paces of
their various transformations.

Enough with the generalities! -- I have been spinning out
on this thread a developing example base of propositional
expressions, both text-strings and data-structural forms,
along with many examples of step-wise transformations on
these expressions that preserve something of significant
import to me, at least, and perhaps someday also to you,
something that I have from time to timbuktu, and back again,
called their "logical equivalence class" (LEC), and then again,
something that I could as well call their "constraint information"
or their "denotative object".  So let me then express my interest in
a family of "constraint information preserving transformations" (CIPT's),
or equally well, "denotative object preserving transformations" (DOPT's).

To focus still more, let us return to that Splendid Theorem noted by Leibniz,
and look at the two very distinct ways of transforming its initial expression
until we arrived, in either event, at an equivalent expression, one that made
its tautologous character or its theorematic nature as evident as it could be.

Just to remind you, here is the Splendid Theorem again:

---------------------------------------------------
 Theorema Praeclarum (Leibniz)
---------------------------------------------------
|
|     r o   o s     o rs
|       |   |       |
|     p o   o q     o pq
|        \ /        |
|         o---------o
|         |
|         |
|         @                   =         @
|
|   ((p(r))(q(s))((pq(rs))))  =      [blank]
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤

The first way of transforming the expression
on the left hand side of the equation was
described as "proof-theoretic" in nature.
Because we have not seen it in a while,
I will repeat the transformation here:

¤~~~~~~~~~¤~~~~~~~~~¤~~~DEQ~~~¤~~~~~~~~~¤~~~~~~~~~¤
|
|     r o   o s     o rs
|       |   |       |
|     p o   o q     o pq
|        \ /        |
|         o---------o
|         |
|         |
|         @
|
=================================================== C1 [Reflect "pq(rs)"]
|
|     r o   o s
|       |   |
|     p o   o q
|        \ /
|      pq o---------o rs
|         |
|         |
|         @
|
=================================================== [Weed "p", "q"]
|
|     r o   o s
|       |   |
|       o   o
|        \ /
|      pq o---------o rs
|         |
|         |
|         @
|
=================================================== C1 [Reflect "r", "s"]
|
|    pqrs o---------o rs
|         |
|         |
|         @
|
=================================================== [Weed "rs"]
|
|    pqrs o---------o
|         |
|         |
|         @
|
=================================================== C3 [Recess "pqrs"]
|
|         o---------o
|         |
|         |
|         @
|
=================================================== I2 [Refold "(())"]
|
|         @
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~QED~~~¤~~~~~~~~~¤~~~~~~~~~¤

The other way of transforming the expression
on the left hand side of the equation was
described as "model-theoretic" in nature.
It began and ended like this:

¤~~~~~~~~~¤~~~~~~~~~¤~~~DEQ~~~¤~~~~~~~~~¤~~~~~~~~~¤
|
|     r o   o s     o rs
|       |   |       |
|     p o   o q     o pq
|        \ /        |
|         o---------o
|         |
|         |
|         @
|
===================================================

...

===================================================
|
|              s o---o---o s
|                 \ /
|                r o---o---o r
|                   \ /
|                  q o---o---o q
|                     \ /
|                    p o---o---o p
|                       \ /
|                        @
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~QED~~~¤~~~~~~~~~¤~~~~~~~~~¤

What we have here are two different styles of "communicative action",
that is, two sequences of signs of the form 's1', 's2', ..., 'sk',
each one beginning with a problematic expression and ending with
a clear expression of the LEC to which each and every sign or
significant expression in the sequence belongs.

Ordinarily, a trajectory through a locus of signs can be taken
to reflect an underlying sign-process, that is, a "semiosis".
So what we have here are two very special cases of semiosis,
and what we might just find it useful to contemplate is how
to characterize them as two species of a very general class.

If that is not too stream of consciousness for you,

Jon Awbrey
