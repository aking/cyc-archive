I would describe things differently, to clarify our misunderstanding.

Following Pinker and Prince, I would start with the idea that "natural
concepts" may include:

bird
dog
game
set of three objects
toad of kind x
pregnant
grandmother of x
number
space
time
...

Some of these natural concepts are based on family resemblances (irregular),
while others have logical definitions (regular).

How we recognize that a referent belongs to a class, depends on the kind of
class for which we are testing membership. E.g. if it is a family
resemblance class, then we may use a cluster-match algorithm to decide if we
are going to call it a "bird". If is a formal class, then we use a formal
definition to decide if we are going to call it "grandmother of x".

To each of these concepts corresponds a label that we use to say a referent
belongs to the class. This label could be called a "predicate", though
normally people only use "predicate" to refer to classes that have formal
definitions.

I agree that the regular, formal classes have definitions within theories.
For example, we may have a theory providing a formal definition of "number",
that corresponds to say, decimals up to a certain length. This theory of
number may be applied in various contexts, and be useful in most situations
that humans reason about, day to day. Another theory may define "number" to
include complex numbers, and be used only in special circumstances.
