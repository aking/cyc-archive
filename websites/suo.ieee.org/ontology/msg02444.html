All,

	I feel the real problem is lack of something to focus on, namely
some starter documents (or whatever we choose to call them).  Given one or
more of them, we can more easily focus discussions and politely ask that
other postings be made elsewhere.   For now, we probably can reduce the
volume as follows.

1.	Please limit postings (to this list) to those dealing with this
project.  General ontology topics should be discussed on the 'Ontology'
list.  See http://suo.ieee.org <http://suo.ieee.org>  for how to subscribe.
This needs to be self-policing, or colleague policing.  If a thread doesn't
belong, announce that 'you' are moving your response to a different list.

2.	If volume in your inbox is a problem, I suggest filtering SUO mail
into a subfolder.

3.	As for alternative mediums of communication (such as Slashdot), if
someone would like to set it up, we can give it a try.

4.	I believe IEEE has a search engine that can search just one site.
I'll look into it.

	These items will probably cut down on the volume, but without one or
more documents to focus on, we won't be making much progress.

	Jim Schoening
