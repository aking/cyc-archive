On Wed, 28 Feb 2001, Yang Yun wrote:
>
> I'd appreciate any way to make it easier to navigate through
> this ontology. I don't use Lisp, and would request that a
> neutral format like XML/MathML be used for interchange.

It's not Lisp.  It's a simplified version of the Knowledge Interchange
Format (KIF -- http://logic.stanford.edu/kif/kif.html) called SUO-KIF
(http://suo.ieee.org/suo-kif.html), which is just (a slightly
eccentric variation of) the language of first-order logic.  It has a sort
of Lispy-looking syntax, but it is an entirely different animal.

Chris Menzel
