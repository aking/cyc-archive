HTML tables are based (loosely) on the old DoD CALS tables that almost all
SGML and XML editors (including WordPerfect) support. The W3C has a
Mathematics Markup Language (MathML) that has been in development for
several years. It's an XML application andis supposed to be accepted as one
of the modules of XHTML (the XML replacement for HTML):
http://www.w3.org/TR/2001/PR-MathML2-20010108/
