Matthew,

On page 101 of this old Linear Algebra text,
out of which I long ago had occasion to teach,
there are four problems involving the variables
"x", "y", "z", "w".

11.   x -  3y -  6z -  3w =   7
    -2x +   y +  4z       =   2
     -x +   y +  3z +   w =  -2
      x -  2y -  4z -  2w =   6

12.   x +  2y             =   5
     3x +  8y +  2z +  4w =  17
    -2x       +  3z +  5w =  -1
      x       -  5z - 11w =  14

15.  2x +  8y + 10z -  6w =  12
     3x + 12y + 19z - 17w =  22
    -2x -  8y -  8z +  5w = -16

16.  4x + 12y +  8z +  4w =  20
     2x +  8y + 12z +  6w =   8
     3x + 13y + 25z +  2w =   5

Here are some questions that I never would have
dreamed to ask my students,  but I am beginning
to reckon that maybe I should have, just to see:

1.  Is the letter "x" in line 1 of Ex. 11
    intended to stand for the same value
    as the letter "x" in line 2 of Ex. 11?

2.  Is the letter "x" in line 1 of Ex. 11
    intended to stand for the same value
    as the letter "x" in line 2 of Ex. 12?

3.  How would you describe the character
    of the symbols that are interpreted
    in the way that "x", "y", "z", "w"
    are interpreted in these Exercises?

4.  What is the nearest natural language
    analogue of these types of symbols?

5.  Specify the interpretive conventions
    that permit you to decide when signs
    like "x", "y", "z", "w" refer to the
    same values and when they do not.
