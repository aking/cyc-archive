Note 114

¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤

Let us now plot out one further complication in the scene
before we proceed with analyzing the design of the action.

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                 o "arbitrary sign for q"  |
|                                /            |             |
|                              sig            |             |
|                              /              |             |
|                 q o---obj---@          computation        |
|                 o            \              |             |
|                 |            int            |             |
|                 |              \            v             |
|                 |               o "canonical sign for q"  |
|                 @                                         |
|                / \                                        |
|               s   t                                       |
|              /     \                                      |
|           X o       o %B%                                 |
|            /         \                                    |
|           s           t                                   |
|          /             \                                  |
|         o---t---o---s---o                                 |
|        /      %B%^k      \                                |
|       /                   \                               |
|      /                     \                              |
|     o                       o                             |
|    c                         p                            |
|                                                           |
o-----------------------------------------------------------o

Here, I have sketched, in what amounts to nothing so novel as
would exceed a variant form of presentation, the old familiar
story of "How the proposition about the world beyond our cave
got itself factored along the lines of a bitwise coding and a
propositional connective" or how the proposition q : X -> %B%
is factored across a composition q = pc, where c : X -> %B%^k
is a coding map and p : %B%^k -> %B% is the boolean function
that corresponds to the propositional connective in question.

As far as I can tell, all of these elements really are present
in one implicit form or another whenever we use a mathematical
style of sentential calculus to reason with propositions about
an external universe of discourse X.

Jon Awbrey
