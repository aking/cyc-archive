so far in my story of juggling we have got the following data:

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                 s  =  symbolic derivative |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o <------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 i  =  image or impression |
|                                                           |
|                                                           |
|                                 s  =  symbolic derivative |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o'<------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 i' =  implemented imagery |
|                                                           |
o-----------------------------------------------------------o

it's a long way from here to anywhere moderately interesting,
as far as actual juggling goes, and i do not really know the
way to get there from here.  but i do have a picture in mind,
if it's not a mirage, that dimly prefigures to me an outline
of a possible station along the way.  it looks a bit like so:

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                                           |
|    Learnable o                                            |
|              |\                                           |
|              | \ !                                        |
|              |  \                                         |
|            ? |   o Knowledge    s  =  "juggling"          |
|              |  /              /                          |
|              | / ?            /                           |
|              |/              /                            |
|    Juggling  o <------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 i  =  <juggling>          |
|                                                           |
o-----------------------------------------------------------o

from now on i will treat the set of letters {"i", "o", "s"}
as variable names, or as the lunatic lover poet may betimes
be licentious enough to say, i will treat the set of things
{i, s, o} as variables, and pray that no one is confused by
what the license permits, say, into thinking what it cannot.

on with the story.  in this episode, i imagine myself to have
reconnoitered my pragmatic niche well enough that I'm able to
tally up a goodly number of the natives and the other regular
inhabitants thereof and to point out many of the more salient
topical features of the surrounding semiotic setting about me.
but enough about me.  let us pass to the legend of the figure.

at this stage of the game, i have come to picture the term of art "juggling"
as a "general name", in effect or in essence, according to your preference,
a name for a generic collection of active performances or a generic class
of virtuosities that a person might possess as a potential disposition.

under the big top of my new frame of mind i have come to view the
whole arena of the juggler's art as a much wider circle than that
instigmatory instance i espied as a child.  in this picture, then,
my cap is now set in the direction of implementing instantiations
enacted by myself within the class named "juggling" that might be
the like of the instance by another actor that i liked as a child.

at any rate, it is customary to let the name of the game called "juggling",
and others like it, to work just a bit equivocally in their thus-arrayed
costumes of address, acting in such a way as to evoke either a general
or a particular denotation, depending on the pending circustances.

in a like fashion, let me use the anglish-bracketed expression "<juggling>"
to denote the sort of "active interpretant" (ai) that forms our especial
concern and interest in the present effort.

the other details that i have placed on the objective wing of the diptych
merely foreshadow the fact that i have begun at this stage to think about
the variety of objective properties that might belong or that must belong
to the act and the art of juggling, no doubt as a factor in the planning
process that would naturally go into acquiring and actualizing the art.

taking five --

jon awbrey
