If you will be gracious enough to float my promissory note on the
issue of definitions by relation-theoretic "universal properties",
then I think that I am finally ready to illustrate the two kinds
of transition from a 2-adic view to a fully 3-adic perspective
on sign relations.  Let us pick one of the 2-adic denotation
tables for color words and work on that, for instance, this:

| green        | "vert"       |
| yellow       | "jaune"      |
| black        | "noir"       |

Let us suppose that Olivier, or someone else of the same name,
does not know any English yet, that Josiane is bilingual with
respect to English and French, and that Kim, with the help of
a French lexicon, is able to get by for easy words like these,
but that Godfrey knows only English.

I think that it tends to become clear rather quickly,
after you spend a while looking at arrays like these,
that this is one of those ideas that may be nice in
principle, and in some ways it is really true that
all of this complexity is actually out there in
the semiotic conduct of sign-using agents, but
that we need another way to get a handle on
the complexities of this activity without
getting utterly overwhelmed by it.

On to plan b.

Option (b) has us add a column for the interpretant sign.

Each of these sign relations has familiar type of structure.
The common object domain is O = {black, green, pink, yellow},
and "at" or "over" each object o in O one finds a "part" of
a "partition", that is to say, an equivalence class of an
equivalence relation on the correspoinding "syntax domain",
which is a convenient name for the set-theoretic union of
the sign domain S and the interpretant domain I, in these
cases having S = I.

For the monolingual interpreters Olivier and Godfrey,
each equivalence class of signs over each object is
in fact just a singletion set, containing the only
name that the agent knows for that object.

For the bilingual interpreters Josiane and Kim+Lex,
the sign relations are completely identical, since
the order of rows does not matter.  The partition
of signs that these two agents share, as a set of
sets, can be written as follows:

|    {{ "pink"   , "rose"  },
|     { "green"  , "vert"  },
|     { "yellow" , "jaune" },
|     { "black"  , "noir"  }}

That should be enough for today.

Jon Awbrey

A.  What is this triadic relational POV, anyway?
    1.  Explain it in terms of 3-adic relation tables.
        Suppose one is given a 2-adic table of objects and signs.
        a.  Option (a).  Add a column for the interpretive agent.
        b.  Option (b).  Add a column for the interpretant sign.
    2.  Aristotle's Approximation to the full-fledged sign relation:
        a.  Assumption (a).  Commonality in the object domain.
        b.  Assumption (b).  Commonality in the interpretant domain.
        c.  Conclusion (c).  Canonically privileged frame of reference.
B.  Bearing of the sign relational POV on several design issues:
    1.  Ontological Integrity versus Ontological Modularity.
    2.  Ontological Invariance versus Ontological Relativity.
    3.  Interoperability among ontological frames of reference.
    4.  Differential topics.
        a.  Change.
        b.  Development.
        c.  Diversity.
        d.  Evolution.
        e.  Inquiry.
        f.  Learning.
    5.  Critical Reflection.
    6.  Dynamics of Inquiry.
        a.  Informational dynamics.
        b.  Psychological dynamics.
        c.  Sociocultural dynamics.
