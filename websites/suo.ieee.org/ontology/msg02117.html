Let us now examine the propositional equation:

¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤
|
|                                      q r
|     q o   o r                         o
|       |   |                           |
|     p o   o p                       p o
|        \ /                            |
|         @              =              @
|
|  (p (q)) (p (r))       =          (p  (q r))
|
|  [p=>q] & [p=>r]       =          [p=>[q&r]]
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤

There are three different ways that come right to mind
right now as to how we might go about formally proving
or systematically checking such a proposed equivalence,
the evidence of whose truth we have still have before
us clearly enough, and in a visually intuitive form,
from the venn diagram that we examined last time.
While we go through each of these ways, though,
let us keep one eye out for the character and
the conduct of each type of proceeding as a
semiotic process, that is, a trajectory, in
this case discrete, through a locus of signs,
in this case propositional expressions, and as
it turns out again in this special sort of a case,
a sequence of transformations that perseveres in the
the denotative object of each expression, that is to say,
the abstract proposition that it expresses, along with the
constraint information about the univeese of discourse that
represents one viable candidate for the informational content
of each expression in the lacing chain of these metamorphoses.

The last couple of sign relations that I presented in detail,
the two that informed the story of A and B, had just eight
triples of the form <o, s, i> in each one.  We have taken
a bit of jump up from that, or turned the 8 on its side,
if you want to look at it that way, since the syntactic
domain, S |_| I, is now a countably infinite multitude.
Actually, as is commonly the case in these computable
examples, S = {S |_| I) = I.  A sign relation that is
of interest to your average logical stakeholder is one
that has for its object domain O the 2^3 = 8 positions
of abstract type B^3 and the 2^(2^3) = 256 propositions
of abstract type B^3 -> B that together make up the form
of the universe of discourse X° that one can specify more
concretely under the rubric of X° = [p, q, r].  With regard
to this universe, taken as a complex but cohering categorical
object, I take the 256 propositions f : B^3 -> B as the objects
denoted by the propositional expressions in S = I, of which our
present example, say g : B^3 -> B, where g = (p (q))(p (r)), is
just one of the contingent members of the whole platonic platoon.
To each of these denotative objects in O there corresonds an entire
LEC of propositional expressions in S = I, each of which denotes it.
Let us now view several distinctive patterns of semiotic process, but
all of which still have in common the circumstance that the sign to sign
transitions do not leave the LEC of expressions within which they originate.

The first example of semiosis is the following proof of the above equation:

¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤~~~~~~~~~¤
|
|                                      q r
|     q o   o r                         o
|       |   |                           |
|     p o   o p                       p o
|        \ /                            |
|         @              =              @
|
|  (p (q)) (p (r))       =          (p  (q r))
|
|  [p=>q] & [p=>r]       =          [p=>[q&r]]
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~DEQ~~~¤~~~~~~~~~¤~~~~~~~~~¤
|
|                    q o   o r
|                      |   |
|                    p o   o p
|                       \ /
|                        @
|
|                 (p (q)) (p (r))
|
=================================================== Double Negation
|
|                    q o   o r
|                      |   |
|                    p o   o p
|                       \ /
|                        o
|                        |
|                        o
|                        |
|                        @
|
|              (( (p (q)) (p (r)) ))
|
=================================================== Collection
|
|                    q o   o r
|                      |   |
|                      o   o
|                       \ /
|                        o
|                        |
|                      p o
|                        |
|                        @
|
|              (p ( ((q)) ((r)) ))
|
=================================================== Double Negation
|
|                       q r 
|                        o
|                        |
|                      p o
|                        |
|                        @
|
|                    (p  (q r))
|
¤~~~~~~~~~¤~~~~~~~~~¤~~~QED~~~¤~~~~~~~~~¤~~~~~~~~~¤

For some reason I always think of that
as the way that our DNA would prove it.
