Dear James,

I see this is a repost, so I will repost my reply.

You have effectively alstered the definition of a base document to that of a
working paper. I do not think that it is useful to devalue the definition of
a base document in this way. The key point is that a base document should:

"base document: Starting point for standards words (it's clear the document
*can* be transformed into standards words). "

To be ready to be turned into standards words the document would have to be
technically complete at least. The current document does not come close to
this.

As I keep saying and will continue to say WE HAVE A USEFUL WORKING PAPER.
Please be grateful for this.
