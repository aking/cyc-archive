Summary of Conjunction

To establish a convenient reference point for further discussion,
Table 49 summarizes the operator actions that have been computed
for the form of conjunction, as exemplified by the proposition J.

Table 49.  Computation Summary for J
o-------------------------------------------------------------------------------o
|                                                                               |
| !e!J  =  uv .     1       + u(v) .    0    + (u)v .   0     + (u)(v) .   0    |
|                                                                               |
|   EJ  =  uv .  (du)(dv)   + u(v) . (du)dv  + (u)v . du(dv)  + (u)(v) . du dv  |
|                                                                               |
|   DJ  =  uv . ((du)(dv))  + u(v) . (du)dv  + (u)v . du(dv)  + (u)(v) . du dv  |
|                                                                               |
|   dj  =  uv .  (du, dv)   + u(v) .     dv  + (u)v . du      + (u)(v) .   0    |
|                                                                               |
|   rJ  =  uv .   du  dv    + u(v) .  du dv  + (u)v . du dv   + (u)(v) . du dv  |
|                                                                               |
o-------------------------------------------------------------------------------o
