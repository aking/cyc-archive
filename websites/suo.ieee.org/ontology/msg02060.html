I have been accumulating this stuff.

What I have is an issues-oriented collection of assertions, mini-essays,
issue statements pro-and-con, rejoinders, rejoinders-to-rejoinders, and
the like.

I've tried to strip out the repetition and only retain the meat.
For example,
I've rewritten what an author said, when he later amended his original
statement
(something backwards, misspelling, ...).

It's in a Microsoft Word document and I haven't held back from using 
footnotes, pop-up comments, outline processor, and such.

I have retained attribution so that you generally know who made an
assetion
and, since some of the context is necessarily gone, I'm sure that there
would
be occasional objections as to the content.

I'm not sure how much merit this has for folks other than myself, but 
I'd be happy to share a copy with anyone who's interested.

It's about .5 megabytes and, for size reasons,
no longer contains the text of the merged ontology.
