 > The last definitional axiom above is contained in the foundational

Correction: The specific nature of the foundational definition of
'SET.LIM$pullback' only ensures that the pullback is Cartesian -- a subset
of the binary Cartesian product. The composition of set functions is defined
by a (slightly complicated) foundational binary function restriction
operator (involves the pullback of opspans, the subconglomeration relation
and ordinary function restriction) -- in short, composition of set functions
is defined to be the binary restriction of composition of class functions.
