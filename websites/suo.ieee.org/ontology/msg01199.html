Disclaimer:  I have been using GML for many years with just
a plain-text editor.  I use a plain-text editor for HTML
and XML, as do a great many webmeisters.  So my endorsements
for any end-user oriented text processing tools are based on
very little personal experience with them.  (My general opinion
is that wysiwyg tools are good for graphics, but for most other
things, the command line is a great improvement over the mouse.)
