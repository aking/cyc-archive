Jon, Matthew, Fred, et al,

	I agree.  At some point the merged ontology has to be able to
represent change.  The question is whether the merged ontology should be
disqualified because the initial, fledgling versions do not represent
change.  I am arguing that, although there are many respects in which the
merged ontology is lacking, it is a foundation upon which we can build
additional content.  Apparently, Matthew and Fred are arguing that, because
the merged ontology has some gaps, we should toss it out and start over from
scratch.

-Ian
