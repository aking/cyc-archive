back to my juggling exercise.

i started out with aristotle's way of explaining the elementary sign relation:
the object is 'pragma', the affect, concept, experience, image, or impression
of the psyche is 'pathema', the sign is 'semeion' or 'symbolon'.  notice that
the initial order of derivation that aristotle lays out is o -> i -> s, where
the sign or the symbol (s) is a fragment of the likeness (i) of the object (o).
thus, the process of interpreting signs and orienting to objects will commonly
be called upon to traverse these components in the reverse order:  s -> i -> o.
if i want to pay attention to the time ordering and have it flow down the page,
then i need to flip my initial diagram of the elementary sign relation like so:

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                 i  =  image or impression |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o--------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 s  =  symbolic derivative |
|                                                           |
o-----------------------------------------------------------o

when we start to think about the way that sign 3-ads are involved in semiosis,
in effect, the various temporal processes that are associated with sign usage,
then we may find ourselves quite naturally notating the sign 3-ad in the form
<o, s, s'> or <o, s_1, s_2> for <o, s, i>.
  
in order to keep from confusing stable characters
with transient roles i assign the following names:

| o^1  =  initial object  =  initial juggling act.
|
| s^1  =  initial sign    =  my impression of o^1.
|
| s^2  =  sequent sign    =  symbolic part of s^1.

using superscripts to index constant names, as with o^1, s^1, s^2,
lets us keep subscripts for components of relations and sequences,
as with o_1, o_2, ..., o_k, or with s_1, s_2, ..., s_k, and so on.

anchoring this to aristotle's account may have created
a little bit of terminological confusion at the outset,
but i think that it will be worth the trouble, in time.

applying all of these new-fangled conventions to our old-fangled example,
we can illustrate the recovery of (something like) an initial impression
from a derivative symbol in the following fashion:

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                 s_1 = image or impression |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o^1 = o--------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 s_2 = symbolic derivative |
| -1-                                                       |
o-----------------------------------------------------------o
|                                                           |
|                                 s_1 = symbolic derivative |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o^1 = o--------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 s_2 = image or impression |
| -2-                                                       |
o-----------------------------------------------------------o

altogether, the story so far can be summed up by stating that we
have a sign relation L = {<o^1, s^1, s^2>, <o^1, s^2, s^1>, ...}.

now, the idea that i might be able to break out of the loop-de-loop
between my initial impression s^1 and my derivative symbol s^2 that
orbit about the original object o^1, and somehow or another be able
to convert the circly circus entire into the implemented imagery of
my own interpretive juggling act o^2, might be depicted as follows:

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                 s_1 = image or impression |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o^1 = o--------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 s_2 = symbolic derivative |
| -1-                                                       |
o-----------------------------------------------------------o
|                                                           |
|                                 s_1 = symbolic derivative |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o^2 = o--------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 s_2 = implemented imagery |
| -2-                                                       |
o-----------------------------------------------------------o

again, easier said than done, but at least this exercise has
served to sort out one or two of the notational obstructions.

jon awbrey
