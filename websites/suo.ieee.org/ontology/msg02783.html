Dear Ian,

> 
> It seems to me that there are two notions of set.  

MW: More particularly what you describe below are two different
notions and should be kept separate. That some may use the same
term for both of them just means you have a homonym problem.

> One is an 
> abstract object
> which has no location in space-time, which has elements that can be
> instances of either Abstract or Physical, and which has an 
> identity that is
> determined exclusively by its elements (if elements are added to or
> subtracted from a set, the set is no longer the same set).  
> Some examples of
> this sort of set are the set of real numbers, the set of 
> registered voters
> in the U.S., etc.  On the other notion of set, which is referred to as
> Collection in the Merged Ontology, a set is comprised exclusively of
> instances of Physical, has a location in space-time, and the 
> identity of the
> set can remain constant even when some members are added or 
> subtracted.
> Some examples of this sort of set are football teams and 
> flocks of geese. 

MW: I would consider these to be organisations (in some general sense) 
rather than sets. In EPISTLE we call them Functional Physical Objects,
because they have a purpose (yes, even a flock of geese).
