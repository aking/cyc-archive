Topic :> Dynamics Of Ontologies 19

o~~~~~~~~~o~~~~~~~~~o~~~~~~~~~o~~~~~~~~~o~~~~~~~~~o

Example 1.  Prognostic Signs And Symptoms (cont.)

The improved theory (Node 4) generates the improved model set (Node 5).

Node 5.  Improved Model -- http://suo.ieee.org/ontology/gif00008.gif

o-----------------------------------------------------------o
| Node 5.  Improved Model                                   |
o-----------------------------------------------------------o
|                                                           |
|   delirium                                                |
|    laughter                                               |
|     (seriousness )                                        |
|      less_dangerous                                       |
|       (more_dangerous ) *                                 |
|    (laughter )                                            |
|     seriousness                                           |
|      (less_dangerous )                                    |
|       more_dangerous *                                    |
|   (delirium )                                             |
|    laughter                                               |
|     (seriousness )                                        |
|      less_dangerous                                       |
|       (more_dangerous ) *                                 |
|      (less_dangerous )                                    |
|       more_dangerous *                                    |
|    (laughter )                                            |
|     seriousness                                           |
|      less_dangerous                                       |
|       (more_dangerous ) *                                 |
|      (less_dangerous )                                    |
|       more_dangerous *                                    |
|                                                           |
o-----------------------------------------------------------o

This output tells us that models with the following conjunctions of
positive and negative features are permitted by the Improved Theory:

1.   delirium  laughter (seriousness) less_dangerous (more_dangerous)
2.   delirium (laughter) seriousness (less_dangerous) more_dangerous
3.  (delirium) laughter (seriousness) less_dangerous (more_dangerous)
4.  (delirium) laughter (seriousness)(less_dangerous) more_dangerous
5.  (delirium)(laughter) seriousness  less_dangerous (more_dangerous)
6.  (delirium)(laughter) seriousness (less_dangerous) more_dangerous

This accords with what we should expect from the aphorism of Hippocrates,
in the case of delirium stipulating that laughter implies less_dangerous,
while seriousness implies more_dangerous, but in the case of no delirium
laying no constraints on the remaining features aside from the fact that
the pairs {laughter, seriousness) and {less_dangerous, more_dangerous}
each have to form logical dichotomies.
