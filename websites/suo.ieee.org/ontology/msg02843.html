All,

	A common netiquette rule is to NOT post "I agree" messages to a
large list.  

	Since we have a volume problem, I suggest we start following this
rule.        

	These messages do have some value, but on a large list are usually
not worth the added volume. If everyone did this, it would be a flood.   

	One alternative is to send a private message to the person you are
agreeing with.  Another is to add value in some way and then reply with your
support.  

Jim Schoening
