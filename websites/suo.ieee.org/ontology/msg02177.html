MW: OK well let me first try to state a method, you will have to 
forgive some lack of formality, but it should not be so unclear what
I mean. Where I come from this method would be informally  known as
reification.

Purpose:

To take any n-adic relation occurence and turn it into n binary 
relations without loss of meaning.

Method:

Create an object (say x) that stands for the n-adic relation as a 
whole.

Take each each element in the relation occurance and create a
relation between it and x. Use the role that the element plays
in the original relation to name this binary relation.

Example:

(<=> (gives John Mary Book1)
     (and (gives x)
          (giver John)
          (receiver Mary)
          (given book1)
     )
)

(probably flakey, but I hope intelligible.)
