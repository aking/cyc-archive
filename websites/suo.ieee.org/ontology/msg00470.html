About this process:

The basic operation of a WG developing a standard is:

        Working Draft Process (roughly)
        Step 1.  Current Working Draft (WDn)
        Step 2.  Receive proposals for changes (working papers and presentations with *
specific* edits defined)
        Step 3.  Committee decides which proposals to adopt.
        Step 4.  Apply edits to WDn to create WDn+1.
        Step 5.  n = n+1
        Step 6.  Go to step 1.
=============================

What happens if at some point, we decide to throw out the existing document and
start afresh. The sensible thing would be to start at N=1 again.  To do otherwise 
suggests a falsehood.
