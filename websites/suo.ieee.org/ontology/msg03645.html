here is a cartoon capsule compendium of my career
in juggling up to this point.  just remember that
i am now using the letters "o", "s", "i" as token
role variables or as "occupant" place-holders for
a motley crew of mutably fleeting fillers thereof:

o-----------------------------o-----------------------------o
|     Objective Framework     |   Interpretive Framework    |
o-----------------------------o-----------------------------o
|                                                           |
|                                 s  =  symbolic derivative |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o <------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 i  =  image or impression |
| -1-                                                       |
o-----------------------------------------------------------o
|                                                           |
|                                 s  =  symbolic derivative |
|                                /                          |
|                               /                           |
|                              /                            |
| juggling act o <------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 i  =  implemented imagery |
| -2-                                                       |
o-----------------------------------------------------------o
|                                                           |
|    Learnable o                                            |
|              |\                                           |
|              | \ !                                        |
|              |  \                                         |
|            ? |   o Knowledge    s  =  "juggling"          |
|              |  /              /                          |
|              | / ?            /                           |
|              |/              /                            |
|    Juggling  o <------------@                             |
|                              \                            |
|                               \                           |
|                                \                          |
|                                 i  =  <juggling>          |
| -3-                                                       |
o-----------------------------------------------------------o

there are, of course, vast gaps in the continuum of juggling's
potential evolution that need to be filled up in the record of
its actual development, the concrete mortar, if you will, that
is needed to render the discrete blocks of inquiry concrescent,
but the best that i can do for the moment is to keep an honest
book of the pages that i've got and the pages that are missing.

still, pasted together in this way, even in such haphazard fashion,
these scattered few pages do begin to tell what i can almost begin
to recognize as a common story, one, that if i were to improvise a
fitting title for its theme, i'd emblem as "easier said than done".
sufficient reason for my motto immanently lies in the elliptic gap
that abides between the logos "juggling" and the physis <juggling>.

the only way that i can imagine to close the gap between the command
and the performance is to return in my mind to the initial images of
juggling virtuosity, to examine the derivative symbols that i formed,
and to see if it is possible to form other derivatives that yield me
a reasonably effective analytic description or sufficiently adequate
reconstruction of the act to be able to emulate it in my own routine.

but there may be other ways.

jon awbrey
