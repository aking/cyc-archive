In addition to E-mail, some face to face meetings may be required.  However, the comment that "E-mail is not enough" is not a technical comment from the perspective of ballot resolution, i.e., unless the proposer submits alternative technical wording, there are no specific technical changes implied by this comment.

My personal feeling is that we would require some face-to-face meetings at some point, but hopefully not too many.  FYI, I have no problem with face-to-face meetings ... but it has been this Working Group's desire/preference to make progress via E-mail.
