Cyc constants which have been added/removed since v0.6b

Current Release: Version 0.7b


E-Mail Comments to: opencyc-doc@cyc.com

The constant names in the "Added Constants" list are those which were not included in OpenCyc 0.6b but are included in OpenCyc 0.7b. The constant names in the "Deleted Constants" list are those which were included in OpenCyc 0.6b but are not included in OpenCyc 0.7b.

   There are special cases in which a constant has been renamed, for example "#$AnnualEventType" in version 0.6 is "#$AnnualTemporalObjectType" in version 0.7. In this case, #$AnnualEventType is listed in the "Deleted Constants" list and #$AnnualTemporalObjectType is listed in the "Added Constants" list. However, these are simply name changes; the constant still exists and can be identified by its GUID regardless of its name. Other examples of name changes are #$ComplexTemporalRelation becoming #$ComplexTemporalPredicate, and #$DefaultDisjointFoodType becoming #$DefaultDisjointEdibleStuffType.
