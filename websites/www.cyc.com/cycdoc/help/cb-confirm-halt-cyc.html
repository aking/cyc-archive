Since unintentionally halting a OpenCyc image will most likely have undesired consequences, this confirmation page has been built into the Halt OpenCyc Image facility.

Clicking on "Yes, Halt System" halts the OpenCyc image. Subsequent OpenCyc images should be started in fresh emacs windows (or in an Xterm window). To start up a new OpenCyc image, follow the instructions in How to Start a New OpenCyc Image.

Clicking on "No" keeps the current OpenCyc image running and returns you to the System Area page.
