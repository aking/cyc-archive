The Constant Diagnostics tool provides the user with an internal, low-level diagnosis of the selected constant.

When the Constant Diagnostics tool is selected, CYC® computes which diagnostics would be appropriate for the given constant, and displays these in a list.

To run diagnostics on the constant, click on the checkbox next to any (every) diagnostic you desire to have run, then press the [Do it!] button. CYC® will run all of the diagnostics you have selected and return the results.

To return the checkboxes to their values before any diagnostics were selected or unselected, press the [Current Values] button.

To cancel the diagnostics completely, press the [Cancel] button.

Detailed descriptions of the different diagnostics and possible results will be available at a later time.
