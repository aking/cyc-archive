Since interface settings for the KB Browser are persistent only for a given CYC® Image, NOT for a given CYC® user, it is often desirable to save a particular user's preferences for later use. This facility allows the user to do just that. To restore options that have already been saved, go back and choose the "Restore Interface Settings" option from the Browser Options page.

Saving User Preferences

To save a file of current interface settings, enter into the input window the filename where the settings should be saved. The pathname for the file should start with the root node of the user's filesystem. For example, user Fred's changes might be saved to the file: 

  /home/Fred/cyc/kb-browser-settings
After entering the filename, press the [Save Settings] button.

To clear the input field, press the [Clear] button.

To restore the pathname to its default value before editing, press the [Reset] button. A default value will not appear the first time a particular user's preferences are saved.

To cancel the save operation and return to the previous page, press the [Cancel] button.

To restore settings in a later CYC® image after having set them here, choose the "Restore Interface Settings" option from the Browser Options page.
