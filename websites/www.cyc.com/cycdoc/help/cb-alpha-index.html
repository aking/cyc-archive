This facility allows the user to search through an alphabetical list of constants.

Searching the Index

The search usually starts with a root node. There are two main ways of selecting this node:

  • A string was entered into the Toolbar input window that CYC® was unable to match with a constant in the KB. In this case the Alphabetical Search is brought up with the phrase "[string] would be here" inserted where [string] would fall in the Alphabetical index were [string] to be a constant name. [string] is then the root node.
  • The "A" icon next to a constant name is clicked. In this case the Alphabetical Search is brought up with the selected constant as the root node.

Constant names beginning with lowercase letters are listed after uppercase ones. Thus "Z" comes before "a".

Click on the "Page Up" link to see the set of constant names immediately previous in the alphabet to the displayed set.

Click on the "Page Down" link to see the set of constant names immediately following in the alphabet to the displayed set.

"Top" takes you to the beginning of the alphabetical listing.

"Bottom" takes you to the end of the alphabetical listing.

Use the list of uppercase and lowercase letters to jump to that part of the alphabet.

As usual, click on a constant name to see that constant displayed in the KB Browser.

Click on the red diamond to view that constant using the Hierarchy Browser.
