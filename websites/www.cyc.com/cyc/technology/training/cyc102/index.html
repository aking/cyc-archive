Cyc 102: Introducing Programming & Inference
Cyc's user interfaces allow the causal end-user to enter, review, and query Cyc's knowledge base without requiring much programming expertise. However, having a better understanding of Cyc inference processes and natural language functionality, allows the user to make better use of Cyc's capabilities. The topics covered in this course include:
  • Opening the hood on the Cyc Inference Engine
  • The Cyc Canonicalizer
  • Explaining Cyc's conclusions using the Inference Browser
  • Natural Language parsing and text understanding
  • Natural Language generation and answer and explanation creation

Prerequisite courses:
  • Cyc 101: Introducing Cyc & OE
Related courses:
  • Cyc 201: Advanced Tools & OE
Follow-on courses:
  • Cyc 202: Advanced Programming & Inference
