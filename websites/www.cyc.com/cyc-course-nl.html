Advanced Cyc Topics

Microtheories: Dimensions of Context-Space

Friday 9:00-10:00 -- Lecture


Overview of Cyc NL

Friday 10:00-11:00 -- Lecture

This talk will introduce natural language processing using the Cyc system. Both understanding (NLU) and generation (NLG) will be covered, with the focus being on NLU. Each component of the system (lexicon, parser, grammar rules, semantic combination rules) will be explained, and we will work through a couple of examples that show how the Cyc knowledge base and inference engine are used to do natural language disambiguation.
