XProc: An XML Pipeline Language

By Zhisheng

XProc is an XML Pipeline language, which is developed by the W3C task group. This XML pipeline language is a useful reference for the design of the pipeline language for the LarKC platform. It is much convenient for us to talk about LarKC pipelines if we use a pipeline language, rather than talk about them in their source codes.

Some interesting features in the XML Pipeline language XProc are:

1) flow control primitives, such as the tags <p:choose>, <p:for-each>, <p:when>, and <p:otherwise>;

2) variable and parameter declarations, such as the tags <p:identity>,<p:parameters>, and <p:variable>;

3) Primary Input and output.
