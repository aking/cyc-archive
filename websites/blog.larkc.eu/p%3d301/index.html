netbeans IDE 6.5 to support groovy

By Quesada

Since java and groovy are the official languages for LarKC, I thought some people may be interested in knowing that netbeans IDE 6.5 will support groovy. I’m not particularly fond of IDEs (vim guy) but they have their uses. The netbeans groovy editor provides syntax highlighting, navigator, code folding, occurrences highlighting, code completion, etc… and some of the best contextual help that I have seen. If you really miss vim shortcuts and modes, there is jvi, a module that implements vim inside netbeans. Even reads .vimrc files.
