Web resources used for communication by/in LarKC

  • project web-site (shop window): http://www.larkc.eu

    • goal: formal picture
    • intended mostly for outside world
    • changing slowly (monthly/weekly)
    • readable by world, writeable by Innsbruck

  • project Wiki (shared knowledge space): http://wiki.larkc.eu

    • goal: internal collaboration and knowledge sharing
    • intended for project members and closely interested 3rd parties
    • readable by world, writeable by consortium members
    • active contributions from all consortium members,
    • this is support infrastructure for running WPs, for sharing information, for management

  • project mailing lists (conversation): https://lists.sti2.at/mailman/listinfo

    • goal: internal discussions, announcements, exchange
    • readable by consortium, writeable by consortium

      • the entire consortium: ict-larkc@lists.sti2.at

      • each WP, eg: larkc-wp1@lists.sti2.at

      • executive mngt. board: larkc-epmb@lists.sti2.at

      • technical mngt. board: larkc-tmb@lists.sti2.at

  • project blog (newspaper): http://blog.larkc.eu

    • goal: share observations about outside world
    • readable by world, writeable by consortium members

  • project document and code repositories, see LarkcProject/HowToSvn

    • GForge: https://svn.gforge.hlrs.de/svn/larkc

      • publication/sharing of open material, code-oriented
      • readable by world, writeable by consortium

    • LarKC: http://svn.larkc.eu/

      • sharing of closed material, paper-oriented
      • readable by consortium, writeable by consortium
