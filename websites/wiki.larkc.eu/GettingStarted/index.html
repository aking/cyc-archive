How do I get started with LarKC ?

LarKC provides the software infrastructure needed to develop large scale applications of semantic computing. Using the LarKC platform, together with a set of plug-ins, allows users to assemble work-flows that identify and assemble the knowledge required to solve a problem, and perform the reasoning required to actually solve it.

While semantic computing promises vastly simplified software development by allowing truly uniform handling of heterogeneous data, and by delegating to inference what would conventionally be done by custom software, in practice it has required developers to produce and integrate a great deal of infrastructure (triple stores, inference engines, data transformers, etc.) It's as if, instead of just using a database or a web-server, in an application, one had to write them from scratch.
