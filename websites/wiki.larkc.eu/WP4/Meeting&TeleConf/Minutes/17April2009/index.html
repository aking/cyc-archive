Demonstration of Reasoner Plug-ins in Early adopter's workshop at ESWC2009 in Crete

Barry's message: we can do something with the prototype rule-based reasoner. It is an example of wrapping an existing reasoner (IRIS), it has a configurable rule-set, so it could be used to explore what differences turning on and off certain rules makes and it would also be possible to experiment with different evaluation strategies. The only downside, is that the current version is a basic prototype and doesn't have the SPARQL to Datalog conversion yet (it is using Jena to do the query answering).

Response: Interesting. Hope that we can see the demonstration of rule-based reasoner at the Milan meeting.
