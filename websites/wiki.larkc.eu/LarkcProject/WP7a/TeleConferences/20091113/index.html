Phone conference

13 November 2009, 15:00 - 16:25 CET

Participants

  • Bo (AZ)
  • Vassil, Todor (Ontotext)
  • Florian (STI)
  • Yi Zeng (WICI)
  • Markus (Siemens)

Agenda

  • Discuss the use case plan
  • Discuss the collaboration activities
  • Check the action items from the previous meeting
  • AOB

Minutes

  • LLD prototype is up and running constant new improvements in the data
  • The plans is to have a task break-out and well defined WP7a teams
  • Define concrete task objectives that will fit the use case needs
  • Task: Personalized search and refinement based on user interest retentions
    • objective: provide result filtering/(ranking?) using interest topics
    • members: WICI (implement the algorithm), AZ (define usage scenarios), Ontotext (implementation and platform integration)
    • milestones: M1 UI mock-up (November), M2 prototype (January)
    • evaluation: initial evaluation by the use case, further evaluation by the researchers using 3 result sets
  • Task: Random indexing semantic search
    • objective: provide better search using the LLD model interpretation

    • members: Sheffield (?), ?AstraZeneca (evaluation), Ontotext (implementation)

    • milestones: deadline is end of March
  • Task: Causative relationships generation
    • objective: find new causative relation between the LLD concepts

    • members: Siemens (learning), ?AstraZeneca (evaluation & more usage scenarios), Ontotext (implementation)

    • milestones: deadline end of March (check WP3 deliverables?)
  • Feedback to the platform to be discussed on the Amsterdam meeting

Action Items

  • Provide a mock-up for web interface (AZ with contribution from Ontotext)

  • Update the document at http://wiki.larkc.eu/LarkcProject/WP7a/Cross%20WP%20collaboration%20ideas (WICI)

  • Organize a phone conference for task 2 between Sheffield, WP7b, WP7a responsible members
  • Organize a phone call between WP7a responsible members, WP3 (Volker)
  • Check state-of-the-art in causative relationships generation (Ontotext/AstraZeneca)

Next call
