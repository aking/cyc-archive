WP presentation structure

Content:

  1. Introduce who you are (1 slide)
  2. Objectives of the WP
  3. scientific achievements (justifies received funding)
  4. ... and how they contributed to the overall vision of the project

  5. Open issues & Next steps (justifies future funding)

All presentations must use the standard template, http://wiki.larkc.eu/LarkcProject/AdminForms

All presentations must use the standard slides/images for:

  1. data layer
  2. data structures (labeled set etc)
  3. rdf graph depiction
  4. etc

    • include traffic predictions in demo
    • traffic aware pathfinding
    • WP3 plug in to do it

    • upscale = from 1 to 300000 data sets; semantic enrichment via selection
    • semantic search on monographs
    • evaluation (this may take longer than 6 months)
