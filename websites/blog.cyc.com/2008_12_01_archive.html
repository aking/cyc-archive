Learn to use OpenCyc, and other LarKCs

The OpenCyc tutorial from ISWC 2008 is now available for your watching pleasure. In this tutorial you'll learn about the content and representation of Cyc, how to understand Cyc's content on the web (like OpenCyc endpoints), how to add knowledge to Cyc, and how to use Cyc web service APIs!

In other news, Cycorp Europe just released its first platform prototype code release to the LarKC consortium. To understand what we hope this will mean for the future of the web, see the LarKC web site. In a few months, you'll be able to get the code too.
