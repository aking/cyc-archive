Semantic Knowledge Source Integration (SKSI) provides a declarative mapping between external sources of structured knowledge and the Cyc Knowledge Base. Mapped knowledge sources can be treated as just another part of the Knowledge Base. This means that mapped knowledge sources can be used in inference, and multiple knowledge sources can be joined entirely ad-hoc without regard to form, protocol, location, or any other such details. SKSI also allows information to be written to a data source by asserting into the knowledge base, again without regard to the details of the information source involved.

With SKSI, Cyc can now be used as a single point of access for all structured data.

SKSI is a declarative interface between Cyc and structured knowledge sources which:

  • treats external knowledge as just part of the Cyc KB
  • enables inference in the external source
  • presents Cyc content and inferences in other forms
  • enables arbitrary joins between arbitrary sources
  • ensures that everything has to be translated just once
  • transforms Cyc into an automated DBA, librarian, research assistant, ...
With SKSI, Cyc is one complete solution for integrating knowledge sources.
